# Benjamin Levy Portfolio

A modern personal portfolio website built for Benjamin Levy.

## Overview

This portfolio presents Benjamin Levy’s personal brand, experience, skills, and professional information through a clean, responsive, and polished web interface.

## Features

- Custom personal branding
- Professional portfolio layout
- Headshot and logo integration
- Responsive design
- Smooth modern frontend structure
- Deployed with Vercel

## Tech Stack

- React
- Vite
- JavaScript
- Tailwind CSS
- Vercel
